1. make sure there is a .pypirc file in the home directory. It is contained
   in this directory
1. make a GitHub release
2. update setup.py file, editing for example the version number to match the
   GitHub repo release
3. create README.rst file if README.md has been updated
   pandoc --from=markdown --to=rst --output=README.rst README.md
4. test upload
   python setup.py sdist upload -r pypitest
   and check on https://testpypi.python.org/pypi/NuGridpy
5. upload for real
   python setup.py sdist upload -r pypitest
   and check on
